<Info>
**GET** https://**your_subdomain**.variable.global/api/pact/2/footprints/**footprint_id**
</Info>

<Note>
  **Only footprints that are valid according to the PACT standard will show in the API, regardless of the sharing
  setting.**

  This generally means footprints that have all of the required fields filled out.
  Details on the PACT standard and required fields can be found [in the PACT
  documentation](https://wbcsd.github.io/tr/2024/data-exchange-protocol-20241024/).
</Note>

<RequestExample>
```bash
curl --request GET \
  --url "https://your_subdomain.variable.global/api/pact/2/footprint/footprint_id" \
  --header "Authorization: Bearer <access_token>"
  --header "Content-Type: application/json"
```
</RequestExample>
